# chuni-penguin

# 概要

Twitter上で動くbotです。CHUNITHMの課題曲出してくれます。

ID : [@chuni_penguin](https://twitter.com/chuni_penguin) 


## 使い方

**#お願いチュウニペンギン**と**楽曲レベル**（現状で指定できるのはこれだけ）を含めてツイートすると**引用RT**で課題が返ってきます。
